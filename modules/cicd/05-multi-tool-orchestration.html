Normalize with SARIF

SARIF is the lingua franca of code scanning. Convert every tool’s output to SARIF and merge for a single, de-duplicated view.

# Merge SARIF from multiple tools
mend sca scan --dir . --format sarif > mend.sarif
# other-tool scan --format sarif > other.sarif
sarif-merge mend.sarif other.sarif -o combined.sarif

💡 One Gate, Many Tools

Aggregate results before gating so a single, consistent policy decides pass/fail, no conflicting gates.

Avoid Gate Conflicts

When multiple scanners can fail a build, define precedence and ownership so pipelines stay predictable.

⚠️ Don’t Stack Redundant Gates

Overlapping tools each blocking independently creates confusing failures. Consolidate into one decision point.
