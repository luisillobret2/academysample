Automated Remediation with Renovate

Renovate opens intelligent update PRs so dependencies stay current without manual effort. Tune it to match your risk appetite.

// renovate.json
{
  "extends": ["config:recommended"],
  "vulnerabilityAlerts": { "labels": ["security"] },
  "prConcurrentLimit": 10,
  "schedule": ["after 10pm every weekday"]
}

Webhooks & API

React to scan events in real time. Webhooks push events to your systems; the API lets you pull data and drive actions.

# Fetch findings for a project via the Mend API
curl -H "Authorization: Bearer $MEND_TOKEN" \
  "https://api.mend.io/v3/projects/$PROJECT_ID/findings?severity=high"

💡 Event-Driven Security

A webhook on “new critical finding” can open a ticket, page on-call, and post to chat, no human polling required.

Route to the Right Team

Automatically assign findings by repository, service owner, or label so nothing sits unowned.

⚠️ Secure Your Automation Credentials

API tokens and webhook secrets are high-value targets. Store them in a secrets manager and rotate regularly.
