Mend in Your IDE

The Mend IDE extension surfaces vulnerabilities, license issues, and fix suggestions inline as you code, available for VS Code, IntelliJ, and other JetBrains IDEs.

  1. Install the Mend extension from your IDE marketplace.
  2. Authenticate with mend auth login or a personal access token.
  3. Open a project; findings appear inline with severity and reachability.
  4. Apply suggested fixes or open the recommended upgrade directly from the editor.

Scanning from the CLI

The Mend CLI runs the same engines locally and in CI, so results are consistent everywhere.

# Scan open-source dependencies
mend sca scan --dir .

# Scan source code (SAST)
mend sast scan --dir .

# Fail locally on high or critical findings
mend sca scan --dir . --fail-on high

Shift Left with Pre-Commit

Catch issues before they ever reach a pull request by gating locally.

# .git/hooks/pre-commit
#!/bin/sh
mend sca scan --dir . --fail-on critical || {
  echo "Critical vulnerabilities found. Commit blocked."
  exit 1
}

💡 Faster Feedback = Cheaper Fixes

A vulnerability caught in the IDE costs minutes; the same issue caught in production costs orders of magnitude more.
