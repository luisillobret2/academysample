Auto-Remediation & Mend Renovate

🕒 40 min 📚 4 sections ⭐ 200 XP Level 2 - Intermediate

Automated Dependency Updates at Enterprise Scale

Finding vulnerabilities is only half the battle — fixing them efficiently is what differentiates mature security programs. Mend provides multiple remediation pathways that integrate directly into developer workflows.

Mend Renovate

Mend Renovate is the industry-leading automated dependency update tool. It keeps open-source dependencies current with intelligent, non-breaking pull requests. Renovate supports 90+ package managers and integrates with all major source code platforms.

🔄

Intelligent PRs

Renovate creates PRs with changelogs, release notes, and compatibility scores. It groups related updates to reduce PR noise and schedules updates based on your team's preferences.

🛠

Configurable Rules

Define rules for auto-merging patch updates, scheduling major version bumps for weekdays, or pinning specific dependencies. Configuration lives in a renovate.json file in your repo.

🏠

Enterprise Scale

Manage Renovate across thousands of repositories with organization-wide presets, centralized configuration, and admin dashboards tracking update adoption.

🤖

Agentic Remediation

Mend SCA feeds vulnerability information with reachability analysis into AI code assistants for rapid remediation of open-source vulnerabilities, directly in the AI workflow.

Renovate Configuration

Renovate is configured through a renovate.json file in the root of each repository. Here’s a typical enterprise configuration:

{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": [
    "config:recommended",
    "group:monorepos",
    ":separateMajorMinor"
  ],
  "schedule": ["before 9am on Monday"],
  "automerge": true,
  "automergeType": "pr",
  "packageRules": [
    {
      "matchUpdateTypes": ["patch"],
      "automerge": true
    },
    {
      "matchUpdateTypes": ["major"],
      "automerge": false,
      "assignees": ["@security-team"]
    },
    {
      "matchPackagePatterns": ["eslint"],
      "groupName": "eslint"
    }
  ],
  "vulnerabilityAlerts": {
    "enabled": true,
    "labels": ["security"]
  }
}

💡 Partner Value Proposition

Mend Renovate is open-source and powers dependency updates for over 1 million repositories globally. The enterprise version adds centralized management, organizational presets, and advanced scheduling — making it easy to manage thousands of repos consistently. This is a strong differentiator against competitors who only offer basic fix PRs.

Remediation Workflows

Beyond Renovate, Mend provides additional remediation pathways:

Method How It Works Best For
Renovate Auto-PR Automated PRs with version bumps, changelogs, and test results Ongoing dependency maintenance
Agentic SCA Fix AI assistant autonomously finds and fixes vulnerabilities pre-commit Immediate fixes during development
Security Fix PR Priority PRs for security vulnerabilities with fix versions identified Critical CVE response
Jira Integration Automatically create Jira tickets for vulnerabilities requiring manual review Complex remediation requiring team discussion
Policy-Driven Blocking Block PRs that introduce new vulnerabilities above severity threshold Preventing new risk from entering codebase
